# vue-graphql

## Project Description
- Using Tailwind css
- Implementation of Graphql using apollo
- Include Server sample using express for graphql server

## Project setup
```
npm install
```

### Run Project
```
npm run serve
npm run apollo:run
```

### Compiles and minifies for production
```
npm run build
```

### Run Apollo Server
```
npm run apollo:run
```
